 🧬 DNA Sequence Checker

A C++ console application that checks if a DNA sequence is valid using Stack and Queue data structures.

 📌 About the Project

This project reads a DNA sequence and checks if the base pairs are correctly matched.
It uses a Queue to read the sequence and a Stack to check if the pairs match correctly.

Valid DNA pairs:
- A pairs with T
- T pairs with A
- C pairs with G
- G pairs with C
  
 ✨ Features

- Read DNA sequence using Queue
- Check DNA pairs using Stack
- Show Stack step by step visually
- Detect invalid characters
- Detect invalid DNA pairs
- Show final result (Valid or Invalid)



 🛠️ Technologies

- C++
- Stack Data Structure
- Queue Data Structure
- Code::Blocks IDE



 ▶️ How to Use

1. Open the project in Code::Blocks
2. Build and run the project (F9)
3. Enter a DNA sequence (e.g. ATCGCGAT)
4. See the result step by step


📝 Notes

- DNA sequence length must be even
- Only A, T, C, G characters are allowed
- The program shows the Stack visually after each step
